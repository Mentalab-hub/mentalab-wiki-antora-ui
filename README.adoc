= Mentalab wiki Antora UI

This repository contains the UI files to package a UI bundle for use with Antora to build the https://wiki.mentalab.com[public Mentalab wiki]. It's modified from the https://gitlab.com/antora/antora-ui-default/[Antora default UI].

To preview the UI with a dummy wiki, refer to the README of the Antora default UI.
